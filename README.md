# Workforce Utilization Analytics Dashboard

## Project Overview
This project focuses on analyzing **employee workforce utilization** across attendance, remote work, and leave patterns for a company. Using **Power BI**, the project transforms raw HR attendance data into actionable insights to support workforce planning and talent management decisions.

The analysis covers **80 employees over a 3-month period (April–June 2022)**, examining trends in attendance, work-from-home utilization, and leave patterns.

---

## Problem Statement
Organizations need a clear understanding of how employees utilize office presence, remote work, and leave to improve workforce efficiency and planning. Key questions addressed in this project include:

- What are the monthly trends in attendance and remote work?  
- How do sick leave and other leave types affect workforce utilization?  
- Are there identifiable patterns in peak office or WFH days?  

---

## Dataset
The dataset contains daily attendance records for employees, including:

- Employee Code & Name  
- Daily attendance status (Present, Work From Home, Paid Leave, Sick Leave, Weekly Off, Holidays)  
- Monthly totals and KPIs  

**Note:** Dataset included is a sample of 80 employees for 3 months.  

File: `Dataset.csv`

---

## Dashboard
The interactive **Power BI dashboard** visualizes:

- Monthly attendance rate  
- Work-from-home trends  
- Sick leave and other leave patterns  
- Mid-month dips in attendance and shifts in peak WFH days  

The dashboard allows stakeholders to explore workforce utilization trends and make **data-driven talent management decisions**.

File: `Workforce Utilization Analytics.pbix`  

![Dashboard Preview](Workforce-Utilization-Analytics/Images/Dashboard.png)

---

## Key Insights
- **Mid-month dips in attendance** observed across all months  
- **WFH utilization increased by the end of June**, indicating flexible work patterns  
- **Shift in peak WFH days from Fridays (Apr–May) to Mondays (June)**, highlighting evolving employee preferences  
- Provides a clear view of workforce behavior to support planning and operational decisions  

---

## Tools & Technologies
- **Power BI**: Dashboard development and visualization  
- **Excel / CSV**: Data cleaning and preprocessing  
- **DAX**: KPI calculations and measures  
- **Data Modeling**: Standardizing HR metrics for consistent reporting  

---

## How to Use
1. Open `Workforce Utilization Analytics.pbix` in Power BI Desktop  
2. Load `Dataset.csv` if not already connected  
3. Explore interactive visuals to analyze workforce trends  
4. Use filters to segment data by month, WFH, leave type, or employee
